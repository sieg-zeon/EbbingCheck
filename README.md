# EbbingCheck

## Setup

```shell
# install pnpm via Volta if you've been not installed it yet.
volta install pnpm

cp .env.default .env

pnpm i
```

## Run

Spin up the Vite dev server:

```shell
pnpm dev
```

Or build your app for production and run it:

```shell
pnpm build
pnpm start
```
